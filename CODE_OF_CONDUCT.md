# Code of Conduct

## Our Standard

We want Foresight CLI to be welcoming to beginners and experienced contributors alike.

Examples of behavior that help build a positive community:

- being respectful in issues, pull requests, and discussions
- accepting constructive feedback
- assuming good intent and asking clarifying questions
- focusing criticism on code and ideas, not people

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory language
- trolling or intentionally disruptive behavior
- publishing someone else's private information

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer privately through the repository contact details once the public repo is live.
